# Интерактивное меню для кафе

Минимальная учебная версия меню кафе на обычных HTML, CSS и JavaScript.

## Что есть в V1

- категории блюд;
- поиск по названию;
- фильтр по цене;
- карточки блюд с фото, ценой и кнопкой заказа;
- адаптация под телефон и планшет.

## Как запустить

Откройте файл `index.html` в браузере.

Если хотите запускать через локальный сервер, можно использовать Python:

```bash
python -m http.server 8000
```

После этого откройте в браузере:

```text
http://localhost:8000
```
